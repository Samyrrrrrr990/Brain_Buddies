# Brain Buddies

Brain Buddies is a student-led non-profit organization focused on providing accessible and high-quality academic support to students from elementary to high school. Our goal is to offer dependable educational help, mentorship, and structured learning resources without financial barriers.

## Mission

Brain Buddies aims to ensure every student has access to academic support that is accessible, reliable, and community driven. We focus on creating an environment where students feel supported, confident, and capable of improving their academic performance and long-term skills.

## What We Provide

1. One-on-one tutoring for subjects including math, science, English, and introductory coding  
2. Structured learning pathways that guide students step by step  
3. Workshops, presentations, and study-focused events  
4. Opportunities for volunteers to mentor, teach, and contribute to community learning  

All tutoring and operations are volunteer-based in support of equitable access to education.

## About This Repository

This repository hosts the static website for Brain Buddies. It includes the complete source code for the public site, including HTML, CSS, JavaScript, static assets, branding materials, and documentation.

The purpose of the website is to introduce our organization, explain our services, provide contact information, and allow students and volunteers to learn more about our work.

## Local Setup

1. Clone the repository  
2. Open index.html in any browser  
3. Modify files as needed and refresh the page to view updates  

No additional dependencies or build steps are required because this is a static website.
